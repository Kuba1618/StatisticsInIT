#ENG

Arithmetic average = sum all elements / amount of those elements

Weighted average = (every single element * his value) / amount of those elements

Median = the value in the middle of a data set

Moda = the most frequent value

Quartiles:

    Example: 5, 7, 4, 4, 6, 2, 8
    Put them in order: 2, 4, 4, 5, 6, 7, 8

    Quartile 1  = 4 (the second number from among all these stuff)
    Quartile 2 (Median) = 5
    Quartile 3 = 7

Qurtozis = a measure of the tailedness of a distribution

Standard deviation = a measure of how dispersed the data is in relation to the mean
